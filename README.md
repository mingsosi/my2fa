# SuMing 私有 2FA 解析与迁移系统 (极密版)

这是一个专门优化并重构的 Google Authenticator 单文件纯离线迁移工具。

## 🌟 核心特性
1. **纯粹的单文件离线**：全套解析算法（jsQR, XZing, Protobuf）被全部打包在一个 `index.html` 中，无需部署后端，支持 `file:///` 协议断网环境下完美运行。
2. **📷 摄像头实时侦测抓取**：对抗部分系统的防截屏限制，新增绿色扫码取景框，一旦对焦瞬间捕获即刻解析，附带提示音反响。
3. **🛡️ 纯净导出支持**：支持导出 Aegis (神盾) 专属的导入格式 JSON，并向下兼容常见的通用 CSV 或原生链接。

## 📥 如何使用
本工具涉及您最核心的二步验证密钥，为了保障纯净和绝对安全，提供以下两种最佳实践：

**【普通安全用法（推荐）】**
1. 克隆或直接下载本仓库中的 `index.html`。
2. 断开您的电脑网络（拔掉网线或关闭 Wi-Fi）。
3. 使用最新版 Chrome / Edge 浏览器双击打开直接使用。
4. 使用完毕后关闭浏览器，点击“清空”或清除浏览历史，连上网络。您可高枕无忧。

**【高度定制用法（服务器外网）】**
可以将本文件放置于带有强制 `HTTPS SSL` 认证的 Nginx 或 Caddy 服务器网页根目录下访问（必须使用 HTTPS，否则浏览器引擎会拒绝调取系统前置/后置摄像头），并通过 `.htpasswd` 进行全站隔离防护。
